== Description ==

Add a custom field (onepix_token) to any post or page, and be notified by email or SMS when it is accessed. 
Can be placed on newsletter signup pages, order confirmation pages, or any other important page that you want to know about.

A free onepix.me account is required.

== Installation ==

1. Upload `wp_onepix` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get your API key from OnePix.me and add it in the Plugin settings.
